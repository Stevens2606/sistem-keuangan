# Sistem Informasi Manajemen Keuangan

Aplikasi manajemen keuangan berbasis web dibangun dengan Laravel 12.

## Fitur
- Dashboard dengan grafik keuangan
- CRUD Transaksi (kas masuk & keluar)
- Manajemen Kategori & Anggaran
- Laporan PDF & Export Excel
- Filter & pencarian transaksi
- Manajemen profil pengguna

## Teknologi
- Laravel 12
- MySQL
- Tailwind CSS
- Chart.js
- DomPDF
- PhpSpreadsheet
